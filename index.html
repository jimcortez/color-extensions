Color Shader Toy - Cool/Warm Contrast

Created for CCSF Art 126 by Jim Cortez. | Source Code Repository | Line Shader ISF Editor

Cool-Warm Foreground/Backgrounds

  • Background Color
  • Foreground Color
  • Middle Ground is gradient starting with greyscale value of the foreground and ending with greyscale value of background.
  • Second pane inverts the foreground/background colors.

    Opposite-Temperatures

    • Base Color
    • Foreground is main color with 20% more Saturation
    • Background is color with opposite temperature as foreground. Calculated with inverse of a & b in the LAB color space.
    • Middle Ground is gradient starting with main color, then shading until 10% of complementary color.
